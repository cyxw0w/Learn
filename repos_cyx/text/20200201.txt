제어문

switch ~  case문 (수식의 결과값으로 선별적으로 처리) - 분기문
switch( val )
{
case 값1: 
	명령어; 
	break;
case 값2: 
	명령어; 
	break;
default:
	명령어;
	break;
}
break는 해당 case로 부터 시작된 명령어가 실행될때 
원하는 위치에 switch문 중단시키는 부분을 지정

반목문

for문 (조건식에 따라 특정 명령 수행하는 조건 반복문)

형식
for(초기식;조건식;증감식)
	명령;

while문(조건식에 따라 특정 명령을 수행하며, 거짓이 될 경우 LOOP를 탈출)

형식
while(조건식)
	명령;
